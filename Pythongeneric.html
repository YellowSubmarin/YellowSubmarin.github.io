Варианты использования f'{}'

Форматирование больших чисел - определение символа разряда.

n: int = 1000000000
print(f'{n:,}')
g: float = 1e3
print(f'{g:,}')

Определения количества заполняющих символов и самого символа до определенного размера строки.

lino: str = 'something'
print(f'{lino:>20}', '|')
print(f'{lino:<20}', '|')
print(f'{lino:^20}', '|')

print(f'{lino:#>20}', '|')
print(f'{lino:*<20}', '|')
print(f'{lino:&^20}', '|')

Определение формта даты.

from datetime import datetime
now: datetime = datetime.now()
print(f'{now:%d.%m.%Y (%H:%M:%S)}')
print(f'{now:%c}')
print(f'{now:%I%p}')

Округление числа.

n: float = 1234.56789
print(f'Result is {n:.2f}')
print(f'Result is {n:.0f}')
print(f'Result is {n:,.3f}')

Определение выражения для отладки.

a: int = 89
g: int = 51
l: str = 'Hello everyone'
print(f'{a + g = }')
print(f'{l=}')
print(f'{bool(a)=}')

Создание каталога и символического линка на него в linux

Автоматизация при доставке кода на пром сервер.

Если код на пром сервер доставляется с помощью Jenkins, то можно использовать yml-файл для создания всех необходимых каталогов на сервере - для логирования, для хранения файлов и т.д. Но в том случае, когда необходимо ввести в работу новый или очередной сервер (при ротации), Jenkins не используется. И тогда на помощь приходят скрипты, а точнее предусмотрительность программиста. 

import os
LOG_DIR = '/ora_data4/Export/IMARKETPLACE/LOG'
HOME_DIR_LOG = '/ora_data1/Scripts/IMARKETPLACE/LOG'
if not os.path.exists(LOG_DIR):
    os.mkdir(LOG_DIR)
if not os.path.islink(HOME_DIR_LOG) and not os.path.exists(HOME_DIR_LOG):
    os.symlink(LOG_DIR, HOME_DIR_LOG)

Полезные однострочные программы

Цикл for

#For loop in one line
mylist = [200, 300, 400, 500]
#Single line For loop
result = [] 
for x in mylist: 
    if x > 250: 
        result.append(x) 
print(result) # [300, 400, 500]
#One-line code way
result = [x for x in mylist if x > 250] 
print(result) # [300, 400, 500]

print(Fib(15)) # 987
